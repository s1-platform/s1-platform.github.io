Maven

S1-Platform is deployed in Maven Central, so you can easily include it into your maven project dependencies: 

<dependency>
    <groupId>org.s1-platform</groupId>
    <artifactId>s1-core</artifactId>
    <version>${s1.version}</version>
</dependency>

Contributing

We are more than happy to accept external contributions to the project in the form of feedback, bug reports and even better - pull requests. At this time we are primarily focusing on improving the user-experience and stability of S1-Platform. Please keep this in mind if submitting feature requests, which we're happy to consider for future versions.

In order for us to help you please check that you've completed the following steps:

  • Made sure you're on the latest version
  • Used the search feature to ensure that the bug hasn't been reported before
  • Included as much information about the bug as possible, including any output you've received, what OS and version you're on, etc.
